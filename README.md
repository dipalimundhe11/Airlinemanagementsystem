# ✈️ Airline Management System
The Airline Management System is a desktop-based application developed using Java Swing. It is designed to manage airline operations such as flight booking, ticket cancellation, and passenger details efficiently through a user-friendly interface.

## 🚀 Features
* Flight Booking System
* Ticket Cancellation
* Passenger Management
* View Flight Details
* Simple and Interactive GUI
* Database Integration

## 🛠️ Technologies Used
* Java (Core Java)
* Java Swing (GUI)
* JDBC (Database Connectivity)
* MySQL / SQLite

## ⚙️ How to Run the Project
1. Clone the repository
2. Open the project in Eclipse or IntelliJ IDEA
3. Configure the database connection (update username and password in code)
4. Run the main class file
